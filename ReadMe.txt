Please follow the following steps to run the project.
1. Open Solution file in visual studio.
2. Run web application and a browser tab will be opened with localhost Url  along with port.
3. Locate ContactManagment collection file from the project folder.
4. Export  ContactManagment collection into postman tool.
5  Replace the localhost Url from the browser into the postman Url and run the API methods from postman.
